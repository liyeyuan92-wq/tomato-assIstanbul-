# 番茄钟GAL智能助手（AI陪伴系统）

一个结合番茄钟、AI聊天、学习管理的一站式陪伴助手，基于Vue3开发，支持移动端适配。

## ✨ 核心功能

1. 🍅 **番茄钟**：自定义时长、工作/休息模式切换、完成提醒
2. 🤖 **AI聊天**：对接DeepSeek API，支持角色情感交互、上下文记忆
3. 📚 **学习模式**：文本/Markdown导入、章节管理、AI内容分析、阅读模式
4. 📝 **便签管理**：本地存储、实时编辑
5. 📅 **每日打卡**：学习时长统计、连续打卡记录

## 🛠 技术栈

- **前端**：Vue3（CDN版）、原生CSS、Axios
- **交互**：响应式布局、CSS动画、模态框管理
- **存储**：localStorage本地持久化
- **API**：DeepSeek大模型接口对接

## 🚀 部署方式

1. 克隆代码到本地
2. 配置DeepSeek API密钥（设置面板）
3. 直接打开index.html即可运行（无需构建工具）

## 💡 核心亮点

- **情感化交互**：根据对话内容自动切换角色表情，提升用户体验
- **异常处理**：API请求失败时自动降级为本地响应，保证可用性
- **性能优化**：定时器内存泄漏防护、本地存储错误捕获
- **移动端适配**：适配375px-1200px全尺寸屏幕
- **智能记忆**：聊天功能支持上下文记忆，在学习模式下自动读取当前章节内容
- **章节编辑**：支持手动编辑学习内容的章节标题和内容
- **通知系统**：使用通知组件替代弹窗，提供更好的用户体验

## 📖 使用说明

### 番茄钟功能
- 点击"开始"按钮启动番茄钟
- 可以自定义工作时长和休息时长
- 完成工作后会自动提醒休息

### AI聊天功能
- 在聊天模式中输入消息与AI角色互动
- AI会根据对话内容自动切换表情
- 支持上下文记忆，能够记住之前的对话内容
- 在学习模式下，AI会自动读取当前章节内容，提供更相关的回应

### 学习模式
- 支持导入文本、Markdown、JSON、PDF、DOC、DOCX文件
- 可以手动编辑章节内容
- 支持阅读模式，提供更好的阅读体验
- AI吐槽功能，对章节内容进行有趣的分析

### 便签管理
- 右侧便签面板支持添加和编辑便签
- 所有便签数据自动保存到本地存储

### 每日打卡
- 点击右上角的日历图标进行打卡
- 查看连续打卡天数和总打卡次数
- 查看学习时长记录图表

## ⚙️ API配置

1. 点击右上角的设置按钮（⚙️）
2. 输入DeepSeek API密钥
3. API端点默认为`https://api.deepseek.com/v1`
4. 点击"测试API"验证配置
5. 点击"保存"完成配置

## 📝 注意事项

- 需要有效的DeepSeek API密钥才能使用AI功能
- 所有数据存储在本地浏览器中，清除浏览器数据会丢失
- 建议定期备份重要的学习内容和角色配置
